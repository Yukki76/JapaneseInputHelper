## ビルド環境
* Visual Studio 2022

## 動作環境
* Windows 10または11
* .NET Framework 4.8.1

## 使い方
* ビルドしたexeファイルを任意の場所に置く。
* お好みでスタートアップに登録していただいても構いません。
* 終了はexeファイルと同じフォルダに「endflag」という空のファイルを作成することにより正常に終了することができます。
* Ctrl+\キーで文字入力のON/OFFができます。
